# Dhaka Imperial College Humanities Club Website

## Project Overview
Official website for Dhaka Imperial College Humanities Club (DIC Humanities Club) - A student-run academic and cultural organization focused on humanities, leadership, research, and social awareness.

## Features
- ✅ Fully responsive design (Mobile-first approach)
- ✅ SEO optimized with proper meta structure
- ✅ Fast loading with optimized assets
- ✅ Accessibility compliant (WCAG 2.1)
- ✅ Contact and membership forms
- ✅ Interactive gallery
- ✅ Executive committee showcase
- ✅ Events calendar and announcements

## Technical Stack
- HTML5
- CSS3 (with CSS Grid & Flexbox)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
